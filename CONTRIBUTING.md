# Contributing

Thanks for your interest in contributing!

## Adding a Tool

To add a CLI/TUI tool to the list, submit a pull request with the following:

1. **Add to the appropriate category** in `README.md`
2. **Follow the table format**: `| [tool-name](url) | Brief description. | Author/Org |`
3. **Ensure the tool meets the criteria**:
   - Has a CLI interface with structured output (JSON, plain text, etc.)
   - Is designed for or works well with LLM-powered agents
   - Bridges AI agents to real-world systems

## Adding a Skill

If the tool has an associated agent skill (SKILL.md), add it to the "Example Skills for Tools in This List" table.

## Guidelines

- Keep descriptions concise (one sentence)
- Include the GitHub URL (or official site if no GitHub)
- Verify the link works before submitting
- Place entries alphabetically within their category
- One tool per pull request makes review easier
